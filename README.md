# Country Catalog Project

## how to use?

## Clone Project 
```bash
git clone https://github.com/PhoSophors/Country-Web-App.git
```
## Open Project

1. Navigate to `Country_Web_App`directory:
```bash
cd Country_Web_App
```


2. Install dependencies:
```bash
npm install
```

3. Start the project:
```bash
npm run start
```
